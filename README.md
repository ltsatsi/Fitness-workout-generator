# Fitness Workout Generator

## Purpose

The application automatically generates personalized weekly workout programs based on 
the user’s goals, available equipment, experience level, and preferred workout frequency. 
It ensures balanced training across all major muscle groups, optimizes recovery, and respects
user-defined constraints such as workout duration and difficulty.

## Tech Stack

- ASP.NET Core MVC
- C# 12
- Entity Framework Core
- SQLite
- Razor Views
- Bootstrap 5

## Core Features

- User profile creation with goals, experience level, equipment
- Algorithm-driven workout generation (no AI/ML)
- Balanced muscle-group rotation
- Progressive overload logic
- Automatic rest-day spacing
- CRUD for exercises and workout plans
- Clean EF Core relationships and navigation properties
